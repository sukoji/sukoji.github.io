# sukoji.github.io

[Seokho Jin](https://sukoji.github.io) — personal site for CV, publications, and blog.

**Live:** https://sukoji.github.io

## Edit content

| Path | What |
|------|------|
| `_data/experience.yml` | Education, work, skills |
| `_data/projects.yml` | Project cards |
| `_data/publications.yml` | Papers |
| `_posts/*.md` | Blog posts |
| `about.md` | About page |

## New blog post

Create `_posts/2025-07-15-title.md`:

```yaml
---
layout: post
title: Post title
date: 2025-07-15
tags: [tag]
---
```

## Local preview
