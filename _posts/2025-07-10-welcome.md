이 사이트는 GitHub Pages + Jekyll로 동작합니다.

## 새 글 쓰는 방법

1. `_posts/` 폴더에 파일을 만듭니다.
2. 파일명 형식: `YYYY-MM-DD-제목.md` (예: `2025-07-10-my-post.md`)
3. 맨 위에 front matter를 넣습니다:

```yaml
---
layout: post
title: 글 제목
date: 2025-07-10
tags: [태그1, 태그2]
---
```

4. 본문은 마크다운으로 작성합니다.
5. `main` 브랜치에 push하면 몇 분 뒤 사이트에 반영됩니다.

## 수정할 페이지

| 파일 | 내용 |
|------|------|
| `index.md` | 홈 |
| `about.md` | 소개 |
| `cv.md` | 이력 |
| `publications.md` | 논문 |
| `_config.yml` | 사이트 제목, 설명, 이메일 |

로컬에서 미리보기:

```bash
bundle install
bundle exec jekyll serve
```

브라우저에서 `http://localhost:4000` 으로 확인할 수 있습니다.
