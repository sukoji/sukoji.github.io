이 사이트는 [GitHub Pages](https://pages.github.com/) + [Jekyll](https://jekyllrb.com/)로 운영됩니다. 이력, 논문, 프로젝트를 정리하고, 여기 블로그에 글을 씁니다.

## 새 글 쓰기

`_posts/` 폴더에 파일을 만듭니다.

```
_posts/2025-07-15-멀티에이전트-평가-메모.md
```

```yaml
---
layout: post
title: 멀티에이전트 평가 메모
date: 2025-07-15
tags: [multi-agent, eval]
---

본문을 마크다운으로 작성...
```

`main` 브랜치에 push하면 몇 분 뒤 [sukoji.github.io](https://sukoji.github.io)에 반영됩니다.

## 콘텐츠 수정

| 파일 | 내용 |
|------|------|
| `_config.yml` | 사이트 기본 정보 |
| `_data/experience.yml` | 학력·경력·기술 |
| `_data/projects.yml` | 프로젝트 목록 |
| `_data/publications.yml` | 논문 목록 |
| `about.md` | 소개 페이지 |

## 로컬 미리보기

```bash
bundle install
bundle exec jekyll serve
```

→ `http://localhost:4000`
